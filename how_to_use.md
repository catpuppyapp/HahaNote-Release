### Dropbox:
#### Create a repo:
- Install and launch HahaNote
- click "Create" to create a note repo
- Choose Dropbox and Authorize dropbox account (if on Android, need allow app running in background and disable battery optimization, else maybe no response after authorized)
- choose a local path to store your files
- set a master password
- click "Submit"
- Create some files, 
- Click sync to encrypt and upload them to remote repo you was set


#### Import a repo:
- click "Import"
- choose 'Dropbox' and authorize
- choose a path to store your files
- enter you master password while you creating repo, wrong master password will decrypt data faild
- click "Submit"

### LocalDir:
#### Create a repo:
- Install and launch HahaNote
- click "Create" to create a note repo
- choose a remote path
- choose a local path to store your files
- set a master password
- click "Submit"
- Create some files and click sync

#### Import a repo:
- click "Import"
- choose the same remote path while you creating the repo
- choose a path to store your files
- enter you master password while you creating repo, wrong master password will decrypt data faild
- click "Submit"
